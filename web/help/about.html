TRILL on SWI SH is a web application for trying probabilistic semantic web reasoning with a web browser on any operating system. It was written by Riccardo Zese and Fabrizio Riguzzi.

SWISH was originally written by Torbjörn Lager as a homage to SWI-Prolog. Jan Wielemaker designed and implemented the present version.

TRILL on SWISH extends SWISH. The application runs on top of the ClioPatria framework.

The TRILL on SWISH source is available from Github. It requires SWI-Prolog 7 installed from the latest GIT.

The SWISH source is available from Github.

ClioPatria is available here and from Github.
