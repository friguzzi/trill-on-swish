The hangout room is to find buddies. If you have a question or want to start a discussion

  1. Create a program or notebook by adding a new tab and selecting the appropriate document type and profile.
  2. Prepare code and queries in this document.
  3. Use Open chat for current file from the top-right bell
  4. Create a chat message, briefly explaining what you want.
  5. Wait for people to join you at the page and discuss the problem.

Please restrict messages placed directly in the hangout to general questions about Prolog or this service. Click the message area again to prepare a message.
